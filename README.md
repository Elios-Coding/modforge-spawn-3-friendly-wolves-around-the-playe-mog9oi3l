# Spawn 3 Friendly Wolves on Crouch

Spawns three tamed, friendly wolves around the player when they start crouching while holding a bone in either hand.

Generated by **ModForge AI**.

- Minecraft: `1.21.11`
- Loader: Fabric `0.17.2`
- Fabric API: `0.141.0+1.21.11`

## Build

```bash
./gradlew build
```

The compiled `.jar` will be placed in `build/libs/`. GitHub Actions builds it automatically on push and publishes it to Releases.
